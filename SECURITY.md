# Security policy

## Executable project trust

PyDesign projects are Python programs. Running a project has the same trust implications as running a local Python script. Worker processes improve reliability and cancellation; they are not a malicious-code sandbox.

Do not run projects from unknown sources until their Python has been inspected. The planned GUI opens untrusted projects in source-only mode until the user grants trust.

## Reports

Until a private security-reporting address is established, do not publish a working exploit or sensitive user data in a public issue. Open a minimal issue requesting a private contact channel.

Supported versions and response targets will be added before the first stable release.
